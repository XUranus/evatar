# Features Overview

Evatar is an intelligent personal assistant system that provides personalized information organization and knowledge management through screenshot sync, AI analysis, memory system, and intent reasoning.

## Feature Matrix

| Feature | Description | Status | Platform |
|---------|-------------|--------|----------|
| [Screenshot Sync](./screenshot-sync) | Android auto-monitors MediaStore, incremental upload to server | Implemented | Android + Backend |
| [AI Analysis](./ai-analysis) | LLM multimodal analysis of screenshots, extracting structured info | Implemented | Backend |
| [AI Chat Assistant](./chat-agent) | RAG-based intelligent dialogue with tool calling and web search | Implemented | Android + Backend |
| [Dynamics System](./dynamics) | Background intent reasoning engine, auto-generates insight articles | Implemented | Android + Backend |
| [Memory System](./memory) | Short/long-term memory management, auto-extraction, decay and dedup | Implemented | Backend |
| [Push Notifications](./push-notifications) | Multi-device broadcast push, Webhook and FCM support | Implemented | Android + Backend |
| [Security](./security) | API auth, SSRF protection, encrypted storage, rate limiting | Implemented | Backend |
